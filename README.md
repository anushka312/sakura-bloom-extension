A mindful productivity tool designed to help you cultivate daily rituals and maintain focus.

## Features

-   **Ritual Tracking**: Build daily habits and watch your garden grow.
-   **Focus Timer**: Customizable Pomodoro-style timer for deep work.
-   **Consistency Board**: LeetCode-style contribution graphs for every ritual.
-   **Bloom Grid**: A visual representation of your daily progress.
-   **Sakura Spirit**: A companion that reacts to your productivity levels.
-   **Chrome Extension Ready**: Designed to work as a browser extension with `chrome.storage` persistence.

## Getting Started

### Local Development

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the development server:
    ```bash
    npm run dev
    ```

### Building for Production

```bash
npm run build
```

### Importing as a Chrome Extension

1.  Run `npm run build`.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode**.
4.  Click **Load unpacked**.
5.  Select the `dist` folder generated by the build process.
6.  *(Optional)* Add an `icon128.png` to the `public` folder and update `manifest.json` if you want a custom extension icon.
